Lhasa - Free Software LHA implementation

Lhasa, Tibet viewed from the Pabonka Monastery, highlighting the Potala Palace. Credit: Wikipedia user Qeqertaq, Copyright: Creative Commons Attribution-ShareAlike 3.0 Unported

Lhasa is a Free Software replacement for the Unix LHA tool, for decompressing .lzh (LHA / LHarc) and .lzs (LArc) archives. The backend for the tool is a library, so that it can be reused for other purposes.

Lhasa aims to be compatible with as many types of lzh/lzs archives as possible. To this end, it has an extensive test suite of compressed archives generated by different tools on different operating systems. It also aims to generate the same output as the (non-free) Unix LHA tool, so that it will act as a drop-in free replacement.

Please report any bugs on GitHub, where you can also find the latest changes to the codebase.

In your favorite OS...

  • Debian, Ubuntu and derivatives: apt install lhasa
  • FreeBSD ports: pkg install archivers/lhasa
  • OpenBSD ports: pkg install lhasa
  • macOS (Homebrew): brew install lhasa
  • NetBSD (pkgsrc): pkgin install lhasa
  • OpenSUSE: zypper install lhasa
  • Void Linux: sudo xbps-install lhasa
